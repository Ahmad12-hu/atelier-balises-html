# Atelier des balises HTML

Mini projet HTML/CSS pour reviser les principales balises HTML avec un layout en Flexbox.

## Description

Ce projet est une page de revision realisee en HTML et CSS. Il presente les balises HTML les plus courantes dans une interface simple et lisible.

La page montre notamment :

- les balises de structure
- les balises de texte
- les listes
- les medias
- les tableaux
- les formulaires

L'objectif est de mieux comprendre le role de chaque balise HTML et la maniere dont une page web est organisee.

## Technologies

- HTML5
- CSS3
- Flexbox

## Structure du projet

- `index.html` : structure principale de la page
- `style.css` : mise en forme et layout avec Flexbox
- `html-guide.svg` : illustration utilisee dans la section media

## Objectifs pedagogiques

- apprendre a reconnaitre les balises HTML courantes
- comprendre la difference entre structure et style
- pratiquer la creation d'un layout avec Flexbox
- mieux lire les relations parent / enfant entre les elements

## Comment utiliser le projet

1. Telecharger ou cloner le projet
2. Ouvrir `index.html` dans un navigateur
3. Lire la page et observer les balises dans le code source
